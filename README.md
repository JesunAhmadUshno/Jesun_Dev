How to Deploy to GitHub Pages

Get an API Key:

Go to Google AI Studio.

Create a new API Key.

Open script.js in this folder.

Paste your key inside the quotes: const apiKey = "YOUR_KEY_HERE";

Upload to GitHub:

Create a new repository on GitHub (e.g., portfolio).

Upload index.html, style.css, and script.js.

Enable Pages:

Go to your Repository Settings.

Click on Pages (sidebar).

Under Branch, select main (or master) and click Save.

Wait about 1-2 minutes. GitHub will give you a link (e.g., jesun.github.io/portfolio).
